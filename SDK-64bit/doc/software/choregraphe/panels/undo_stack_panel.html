Undo stack panelÂ¶

The Undo stack panel allows you to see the complete undo/redo stack.

It eases navigation into the different steps that lead to the current project state, in the current Choregraphe session.

Click onto an action displayed in the Undo stack panel to instantly update current project to the selected state.

Warning

A change applied from any state of the current project will erase all other changes made since this state.
