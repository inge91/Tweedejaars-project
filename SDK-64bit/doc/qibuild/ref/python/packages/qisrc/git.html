qisrc.git – Git related tools¶

class that wrap git

qisrc.git.Git¶

class qisrc.git.Git(git_work_tree, git_dir=None)¶
Store a list of ref
for each ref:
  • tracking ref
  • actions
  • status
add(*args)¶

git add

checkout(*args)¶

git checkout

clean(*args)¶

clean working dir

examples:

# remove files and directories
git.clean("--force", "-d")

# also delete files matching .gitignore
git.clean("--force", "-d", "-x")
clone(*args)¶

git clone

commit(*args)¶

git commit

describe()¶

Call describe to get a nice version number

fetch(*args)¶

git fetch

get_current_branch()¶

return the current branch

get_current_ref(ref='HEAD')¶

return the current ref git symbolic-ref HEAD else: git name-rev –name-only –always HEAD

get_current_remote_url()¶

use the origin of the current branch git config –get branch.master.remote

init(*args)¶

git init

is_clean(untracked=True)¶

Returns true if working dir is clean. Nothing to commit, no untracked files .. warning:: there could be a lot of false negatives ...

is_valid()¶

Check if it’s a valid git repo. return False if not valid

list_branch_and_tracking()¶

return a dict with (k, v) = (refs, remote_refs)

local_branches()¶

list all local branches

pull(*args)¶

git pull

push(*args)¶

git push

reset(ref, *args)¶

Reset to the given ref.

Call fetch first just to be sure

set_remote(url, name)¶

set a git remote

set_tracking_branch(current_branch, remote_branch, remote=None, url=None)¶

Set the current tracking branch. If remote is not “origin”, an new url must be given

Useless with git>1.7 ?

show_ref(*args)¶

parse each refs of a repository and return a dict

status(*args)¶

just call git status

sync()¶

sync all branch of a repo

tag(*args)¶

If tag is not signed, just tag it. Else, call tag -s.

Note: if you want to sign your tag, be sure that ~/.gitconfig containg the same user.email and user.name as in your gpg key.

Tag message will always be the name of the tag. (KISS)

try_pull_rebase()¶

do the pull, but call rebase –abort when the pull fail

update_only_ff(local_ref, remote_ref)¶

try to update a local_ref against a remote_ref, if that the merge wont apply => do nothing
