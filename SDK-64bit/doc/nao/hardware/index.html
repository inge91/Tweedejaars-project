Motherboard¶

NAO V. 4.0¶

  • ATOM Z530 1.6GHz CPU
  • 1 GB RAM / 2 GB flash memory / 4 to 8 GB flash memory dedicated to user purposes

NAO V. 3.x¶

  • x86 AMD GEODE 500MHz CPU
  • 256 MB SDRAM / 2 GB flash memory
