Chest Button¶

Bumper¶

  • 2 x bumpers located at the tip of each foot. These are simple ON/OFF switches, thus 4 in total.
Bumper name X(m) Y(m) Z(m)
BumperLFootLeft 0.0876 0.0190 -0.0353
BumperLFootRight 0.0876 -0.0140 -0.0353
BumperRFootLeft 0.0876 0.0140 -0.0353
BumperRFootRight 0.0876 -0.0190 -0.0353

Tactile Head¶

  • 3 X head capacitive sensor
Tactile name X(m) Y(m) Z(m)
TactileHeadFront 0.0312 0.0000 0.1014
TactileHeadUp 0.0010 0.0000 0.1099
TactileHeadBack -0.0257 0.0000 0.1045

Tactile Hand¶

  • 3 X hand capacitive (for each hand)
Tactile Hand name X(m) Y(m) Z(m)
TactileLHandLeft 0.0320 0.0250 0.0030
TactileLHandUp 0.0380 0.0000 0.0250
TactileLHandRight 0.0350 -0.0250 0.0035
TactileRHandLeft 0.0320 0.0250 0.0030
TactileRHandUp 0.0380 0.0000 0.0250
TactileRHandRight 0.0350 -0.0250 0.0035
