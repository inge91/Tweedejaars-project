Detailed Description

Various macro for qi. (deprecated, export API, disallow copy, ..)

  • import/export symbol (QI_IMPORT_API, QI_EXPORT_API)
  • mark function and header as deprecated (QI_DEPRECATED_HEADER, QI_API_DEPRECATED)
  • generate compiler warning (QI_COMPILER_WARNING)
  • disallow copy and assign (QI_DISALLOW_COPY_AND_ASSIGN)
