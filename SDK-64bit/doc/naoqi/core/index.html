NAOqi Core¶

Core modules¶

NAOqi comes with a list of core modules that are always available.

Every module comes with a list of default methods.

You can read the API that is shared by every module in the ALModule API section.

A few general purpose modules are also available by default.

  • ALBehaviorManager starts and stops behaviors
  • ALBonjour advertises NAOqi on Bonjour
  • ALMemory gets and inserts data for every other module to use
  • ALModule ALModule bound methods
  • ALPreferences reads and saves settings from configuration files
  • ALResourceManager handles resources
  • ALLauncher (deprecated) loads dynamic libraries inside NAOqi process and runs executables
  • ALLogger (deprecated) logs information on the robot from a remote machine
