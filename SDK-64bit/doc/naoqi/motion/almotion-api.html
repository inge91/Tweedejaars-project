ALMotion API¶

Overview | API | Motion - advanced

ALMotion provides methods that help make NAO move. It contains commands allowing you to manipulate joint stiffness, joint angles, and a higher level API allowing you to control the walk.

Method Group list¶

As any module, this module inherits methods from ALModule API. It also has the following specific methods:
