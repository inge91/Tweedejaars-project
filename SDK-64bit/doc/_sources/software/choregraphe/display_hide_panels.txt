
Displaying panels
==================

.. _choregraphe-howto-display-hide-panels:

How to display/hide panels
--------------------------

From the View menu in the Choregraphe menu bar, click on the panel that has to be displayed or hidden.

.. image:: /medias/desktop/choregraphe/view_menu.png
   :height: 304 px
   :width: 232 px

Each panel that is already displayed is marked.


.. _choregraphe-howto-move-panels:

How to move panels
------------------

Each panel can be moved so that the interface can be reorganized according to your convenience.

Moving a panel can be done as followed:

#. Click on the top space of the panel where the name is written.

   .. image:: /medias/desktop/choregraphe/move_panels.png
      :height: 78 px
      :width: 441 px

#. Drag the window wherever you want in or outside the interface.

   According to the position of your mouse, the interface will automatically
   propose possibilities to place the panel.

   .. image:: /medias/desktop/choregraphe/move_panels_propositions.png
      :height: 436 px
      :width: 498 px

   A panel can be placed under another, in this case the interface will create a
   tab to access to the background panel:

   .. image:: /medias/desktop/choregraphe/move_panels_tab.png
      :height: 514 px
      :width: 446 px

   A panel can be left as a floatable panel:

   .. image:: /medias/desktop/choregraphe/move_panels_floatable.png
      :height: 552 px
      :width: 521 px
