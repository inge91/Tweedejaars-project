What it does
------------

This module allows you to load dynamicaly libraries or import python modules into naoqi, run executables, run scripts, and check if modules are loaded...

.. warning::

  This module is deprecated.
  
  To create and start your own module: 
  
  * use the methode described in the following section: :ref:`How to create a basic project <create-modules>` or 
  
  * use the `libqi API reference <../../ref/libqi/index.html>`_ with qi::os::spwan*.
